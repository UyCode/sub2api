## Endpoint

```http
POST https://YOUR_DOMAIN/v1beta/models/gemini-3.1-flash-image-preview:generateContent
```

## 请求示例

<CodeGroup>

```bash cURL
curl -X POST "https://YOUR_DOMAIN/v1beta/models/gemini-3.1-flash-image-preview:generateContent" \
  -H "Authorization: Bearer $SUB2API_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [{
      "parts": [{
        "text": "Create a clean 2K product hero image for an API gateway dashboard, premium, realistic, soft teal light."
      }]
    }],
    "generationConfig": {
      "responseModalities": ["IMAGE"],
      "imageConfig": {
        "aspectRatio": "16:9",
        "imageSize": "2K"
      },
      "thinkingConfig": {
        "includeThoughts": false
      }
    }
  }'
```

```python Python
import requests

url = "https://YOUR_DOMAIN/v1beta/models/gemini-3.1-flash-image-preview:generateContent"
payload = {
    "contents": [{
        "parts": [{
            "text": "Create a clean 2K product hero image for an API gateway dashboard."
        }]
    }],
    "generationConfig": {
        "responseModalities": ["IMAGE"],
        "imageConfig": {"aspectRatio": "16:9", "imageSize": "2K"},
        "thinkingConfig": {"includeThoughts": False},
    },
}

response = requests.post(
    url,
    headers={"Authorization": "Bearer SUB2API_API_KEY"},
    json=payload,
    timeout=360,
)
response.raise_for_status()
data = response.json()

image_url = None
for part in data["candidates"][0]["content"]["parts"]:
    file_data = part.get("fileData")
    if file_data and file_data.get("fileUri"):
        image_url = file_data["fileUri"]
        break

print(image_url)
```

</CodeGroup>

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| `contents[].parts[].text` | string | 是 | - | 文本提示词。建议写清主体、构图、风格、用途、负面约束和文字内容。 |
| `generationConfig.responseModalities` | array | 是 | - | 通常为 `["IMAGE"]`，需要文本说明时可用 `["TEXT","IMAGE"]`。 |
| `generationConfig.imageConfig.aspectRatio` | string | 否 | `1:1` | 输出宽高比，如 `1:1`、`16:9`、`9:16`、`4:3`、`3:4`、`21:9`。 |
| `generationConfig.imageConfig.imageSize` | string | 否 | `1K` | `512`、`1K`、`2K`、`4K`。 |
| `generationConfig.candidateCount` | integer | 否 | `1` | 候选结果数量。生产建议先按单图处理。 |
| `generationConfig.responseMimeType` | string | 否 | - | 响应 MIME 偏好，通常不需要手动设置。 |
| `generationConfig.thinkingConfig.includeThoughts` | boolean | 否 | `false` | 是否返回思维过程文本。生产通常关闭。 |
| `systemInstruction` | object | 否 | - | 系统级指令，用于统一风格、品牌规则或安全边界。 |
| `safetySettings` | array | 否 | 官方默认 | 内容安全策略。仅在明确理解影响时调整。 |
| `tools` | array | 否 | - | Gemini 工具配置；图片生成场景通常不需要。 |

<Note>
  如果响应包含多个 `parts[]`，请遍历查找带 `fileData.fileUri` 的图片 part。兼容历史或官方直返场景时，也可以兜底识别 `inlineData.data`。
</Note>

## 响应示例

```json
{
  "candidates": [
    {
      "content": {
        "parts": [
          {
            "fileData": {
              "mimeType": "image/png",
              "fileUri": "https://YOUR_DOMAIN/assets/images/req_abc123/nano-banana-2-0.png"
            }
          }
        ]
      },
      "finishReason": "STOP"
    }
  ],
  "usageMetadata": {
    "promptTokenCount": 10,
    "candidatesTokenCount": 258
  }
}
```

URL 形式响应更适合生产系统保存、转存和分发。base64 兼容字段 `inlineData.data` 会使响应体更大，2K / 4K 场景尤其不建议默认依赖。
