DeepToken Image Gateway 是面向客户的 OpenAI / Gemini 官方能力代理与中转服务。你可以用熟悉的 OpenAI Images 或 Gemini Native 协议接入 `gpt-image-2`、`gemini-3.1-flash-image-preview` 和 `gemini-3-pro-image-preview`，同时获得统一鉴权、账号调度、按次计费和 URL 形式图片返回。

文档中的请求域名使用 `https://YOUR_DOMAIN` 占位，实际调用时替换为你的 DeepToken 中转站域名。

## 接入路径

<Tabs>
  <Tab title="OpenAI Images">
    `gpt-image-2` 使用 OpenAI Images 路由。

    ```bash
    POST https://YOUR_DOMAIN/v1/images/generations
    POST https://YOUR_DOMAIN/v1/images/edits
    ```
  </Tab>
  <Tab title="Gemini Native">
    Nano Banana 2 / Pro 使用 Gemini 原生 `generateContent` 路由。编辑图不是独立 `/edits` 端点，而是在 `contents[].parts[]` 中加入图片 part。

    ```bash
    POST https://YOUR_DOMAIN/v1beta/models/gemini-3.1-flash-image-preview:generateContent
    POST https://YOUR_DOMAIN/v1beta/models/gemini-3-pro-image-preview:generateContent
    ```
  </Tab>
</Tabs>

## 模型速查

| 产品名 | 实际模型 ID | 接口族 | 生成 | 编辑 | 计费展示 |
| --- | --- | --- | --- | --- | --- |
| GPT-Image-2 | `gpt-image-2` | OpenAI Images | `/v1/images/generations` | `/v1/images/edits` | 1K / 2K / 4K 按成功图片次数扣费 |
| Nano Banana 2 | `gemini-3.1-flash-image-preview` | Gemini Native | `:generateContent` | `:generateContent` + `inlineData` | 512px / 1K / 2K / 4K 按成功图片次数扣费 |
| Nano Banana Pro | `gemini-3-pro-image-preview` | Gemini Native | `:generateContent` | `:generateContent` + `inlineData` | 1K / 2K / 4K 按成功图片次数扣费 |

<Warning>
  当前仅支持 GPT-Image-2、Nano Banana 2、Nano Banana Pro。`Nano Banana 2` 和 `Nano Banana Pro` 是产品展示名，API 请求中必须使用实际模型 ID，例如 `gemini-3.1-flash-image-preview` 或 `gemini-3-pro-image-preview`。
</Warning>

## 返回图片

DeepToken 对图片输出统一推荐 URL 形式。OpenAI Images 示例使用 `response_format: "url"`；Gemini Native 响应中优先读取 `fileData.fileUri`。URL 形式可以让客户端更快拿到结果，也能避免 base64 体积膨胀约 33% 带来的传输时间和中转带宽浪费。
